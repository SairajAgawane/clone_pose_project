# 🤖 Clone Tracker – AI-Powered Neon Skeleton Tracking

A futuristic real-time human pose tracking system built using **OpenCV**, **MediaPipe**, **NumPy**, and **Matplotlib 3D**. The application captures live webcam input, detects human body landmarks, generates a mirrored AI clone, visualizes a glowing neon skeleton, and renders a synchronized 3D pose model.

## 🧠 How It Works

### 1. Pose Detection

MediaPipe identifies body landmarks from webcam frames.

### 2. Clone Generation

Landmarks are mirrored horizontally to create a digital clone.

### 3. Neon Rendering

Custom gradient colors and pulse animations generate a cyberpunk effect.

### 4. Motion Trails

Previous skeleton positions are stored and rendered as fading trails.

### 5. 3D Visualization

The pose is simultaneously displayed in a live 3D environment.
